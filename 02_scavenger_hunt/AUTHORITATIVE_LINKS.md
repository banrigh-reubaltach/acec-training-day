# Authoritative Sources — CMMC Scavenger Hunt

**These are the only acceptable source categories for this activity. No blogs. No vendor sites. No AI-generated summaries.**

---

## CUI Program & Registry

**NARA CUI Registry**
https://www.archives.gov/cui

The official registry of all CUI categories and subcategories, including marking guidance and safeguarding authorities.

**Executive Order 13556 — Controlled Unclassified Information**
https://www.federalregister.gov/documents/2010/11/09/2010-28360/controlled-unclassified-information

The executive order that established the CUI program.

---

## CMMC Program

**32 CFR Part 170 — CMMC Program Final Rule**
https://www.ecfr.gov/current/title-32/subtitle-A/chapter-I/subchapter-D/part-170

The codified federal regulation defining CMMC levels, assessment requirements, and implementation timeline.

**DoD CIO CMMC Website**
https://dodcio.defense.gov/CMMC/

The official DoD Chief Information Officer page for CMMC program information, resources, and updates.

---

## NIST Standards

**NIST SP 800-171 Rev 2 — Protecting CUI in Nonfederal Systems**
https://csrc.nist.gov/publications/detail/sp/800-171/rev-2/final

The 110-control security requirements framework that CMMC Level 2 is based on.

**NIST SP 800-171A — Assessing Security Requirements for CUI**
https://csrc.nist.gov/publications/detail/sp/800-171a/final

Assessment procedures for evaluating implementation of SP 800-171 controls.

---

## DFARS Clauses

**DFARS 252.204-7012 — Safeguarding Covered Defense Information and Cyber Incident Reporting**
https://www.acquisition.gov/dfars/252.204-7012

The clause that requires contractors to implement NIST 800-171 and report cyber incidents within 72 hours.

**DFARS 252.204-7021 — Cybersecurity Maturity Model Certification Requirements**
https://www.acquisition.gov/dfars/252.204-7021

The clause that requires contractors to achieve and maintain the CMMC level specified in the contract.

**DFARS 252.204-7020 — NIST SP 800-171 DoD Assessment Requirements**
https://www.acquisition.gov/dfars/252.204-7020

The clause requiring contractors to have a current NIST 800-171 assessment on file in SPRS.

---

## SPRS

**Supplier Performance Risk System (SPRS)**
https://www.sprs.csd.disa.mil/

The DoD system where contractor self-assessment scores are recorded and verified.

---

## Usage Note

If you find an answer on a site not listed above, verify it against one of these primary sources before submitting. The goal is to learn where the authoritative information lives — not to find the fastest Google result.
